## Empire Structure

### Revenue Stream #1: Spark Pro (SaaS)
**Lead**: Stark | **Support**: Sherlock, Jarvis
**What**: Premium stock analysis platform with AI signals
**How it earns**: Monthly subscription ($29/mo Basic, $79/mo Pro, $199/mo Fund)
**Features**:
- Real-time signal alerts (EMA crossover, RSI, Golden Cross)
- AI-powered research reports (Sherlock)
- Portfolio monitoring with smart groups
- Backtesting engine
- API access (Pro+)

### Revenue Stream #2: Agent API
**Lead**: Robin | **Support**: Jarvis, Stark
**What**: AI agent capabilities as a paid API
**How it earns**: Usage-based pricing ($0.01/call basic, $0.05/call premium)
**Endpoints**:
- `/analyze` — Stock/company analysis
- `/research` — Deep research on any topic
- `/knowledge` — RAG-powered Q&A over custom data
- `/signal` — Technical signal detection
- `/summarize` — Document/URL summarization

### Revenue Stream #3: Panya Cloud (Knowledge SaaS)
**Lead**: Jarvis | **Support**: Stark
**What**: Personal AI knowledge base — like Notion + AI memory
**How it earns**: Freemium ($0 free/100 notes, $19/mo Pro, $49/mo Team)
**Features**:
- Auto-learning from conversations
- Knowledge graph visualization
- Pattern detection
- Entity extraction
- Semantic search
- Team knowledge sharing

### Revenue Stream #4: AI Consulting / Custom Builds
**Lead**: Robin | **Support**: All
**What**: Custom AI solutions for businesses
**How it earns**: Project-based ($5K-50K per project)
**Services**:
